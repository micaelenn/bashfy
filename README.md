# :racehorse: bashfy

A collection of Bash scripts designed to streamline local setups on Linux

## wpsetup.sh

This automation simplifies repetitive WordPress setups by downloading the latest WordPress version, setting up virtual hosts on Apache, and creating a MySQL database with minimal manual input.
